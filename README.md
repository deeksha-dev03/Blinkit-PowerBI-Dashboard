# Blinkit Sales & Operations Dashboard (Power BI)

## Project Overview
This project is an end-to-end Power BI dashboard inspired by Blinkit, designed to analyze sales performance, customer behavior, and delivery efficiency. The dashboard focuses on business KPIs, interactive filters, and a clean, Blinkit-themed UI.

## Key KPIs
- Total Sales
- Total Orders
- Average Order Value (AOV)
- On-Time Delivery Percentage

## Dashboard Visuals
- Monthly Sales Trend
- Sales by Customer Segment
- Category-wise Sales Analysis
- Delivery Status Distribution
- Area-wise Sales Performance

## Filters Used
- Month (Multi-select)
- Customer Segment
- Delivery Status
- Area / City
- Payment Method

## Tools & Skills
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX Measures
- Data Modeling
- Dashboard UI/UX Design

##  Business Insights
- Identified peak sales periods and demand patterns
- Analyzed revenue contribution by customer segments
- Evaluated delivery performance to highlight delays
- Compared category and area-wise sales performance
